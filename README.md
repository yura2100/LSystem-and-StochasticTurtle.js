# LSystem and Stochastic turtle.js

Implementation of LSystem with opportunities to make random substitutions and simple "stochastic" canvas turtle.
Example of it work:
![Tree](https://github.com/yura2100/LSystem-and-StochasticTurtle.js/raw/master/example.jpg)
LSystem and Turtle can be used either separately, both together. All methods are described in interfaces. In index.html you can generate some fancy trees yourself.
